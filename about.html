about codejab

codejab is a tiny snippet cms/script built using php, mysql, css, and javascript with elite jquery tweaks.

This project started on wordpress 3.1 RC2 with a stripped down admin, and wp-syntax highlighting plugin, it worked well for what I wanted, but I felt it was time to move it to its own lightweight cms/script.

list of features for codejab:

  1. ajax front end with regards to loading content
  2. clean and simple interface for posting code by upload or cut and paste
  3. syntax highlighting with geshi
  4. ability for code to expire by date (with jquery datepicker)
  5. ajax login/register process with status messages
  6. favorite system for registered users
  7. user submission system (must be approved to submit code)
  8. categories and tags
  9. tag cloud based on posts
  10. pagination system for page(s) of snippets (frontend and admin)
  11. autocomplete search implementation
  12. download of each snippet (.php for php, .pl for perl, etc)
  13. download source code in .pdf format
  14. download zip of code snippet
  15. printing of source code with color highlighting
  16. download zip archive of all source code
  17. thumbs-up rating system
  18. slide down share box for sharing code
  19. embed code for external sites

credits to the scripts that were used in the making of codejab:

  1. Icons for admin panel
  2. jQuery UI (autocomplete and datepicker)
  3. timerpicker (addon for jQuery UI datepicker)
  4. jQuery jqprint (printing source code)
  5. jQuery Form (login/signup)
  6. GeSHi Syntax Highlighter (for all source code snippets)
  7. dompdf (for generating pdfs)
